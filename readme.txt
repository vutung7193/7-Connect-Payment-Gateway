== Description ==
integrated 7 connect(philseven) payment gateway for woocommerce
1. activate plugin in Plugins -> Installed Plugins
2. go Woocommerce -> Settings -> Payment Gateways and choose philseven

== Installation ==
1. Upload folder \'philseven\' into wp-content/plugins
2. Activate plugin in Plugins -> Installed Plugins
3.go Woocommerce -> Settings -> Payment Gateways and choose philseven
